### Week 1 Achievements:

- Cloud Fundamentals & AWS Ecosystem:

* Gained a comprehensive understanding of core AWS service groups, including Compute (EC2), Storage (EBS), Networking (VPC), and Database (RDS).

* Successfully initialized and secured an AWS Free Tier account for development purposes.

- Identity & Access Management (IAM) Mastery:

  * Implemented security best practices by creating a dedicated IAM User with tailored permissions instead of using the Root account.

  * Mastered the management of Security Credentials, including the secure generation and deployment of Access Keys and Secret Access Keys.

- Successfully installed and configured the AWS CLI environment on a local machine.

- Advanced Networking Configuration (VPC):

* Deeply understood VPC architecture, including the roles of Internet Gateways (IGW), Route Tables, and Subnets.

* Gained hands-on experience in Subnetting and CIDR calculation; successfully resolved CIDR overlap errors by strategically allocating non-overlapping IPv4 ranges (e.g., /28 blocks).

* Configured routing logic to distinguish between Public and Private subnets, ensuring secure traffic flow via Route Table associations.

- Operational Integration:

* Demonstrated the ability to synchronize operations between the AWS Management Console and CLI for real-time resource monitoring and management.
